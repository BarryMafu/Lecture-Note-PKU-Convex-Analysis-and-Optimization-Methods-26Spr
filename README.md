# 课程笔记 (PKU 凸分析与优化方法 26Spr)

林宙辰教授25-26学年春季学期凸分析与优化方法课程的个人笔记。 

如果这份笔记可以帮助到您，我倍感荣幸~！也请您顺手给个**star**:star2:！

### 写在前面
我是25–26学年春季学期选修林老师课程的学生。可以看出，林老师为这门课程做了非常充分的准备，教学内容详实、体系完整。不过，相应地，该课程对前置数学知识的要求较高，作业量也确实不小，学生无论是学习新课还是完成作业，都需要投入较多的时间和精力。

此外，在备考过程中，我发现这门课程**缺少一套系统性的往年试题**，这给复习带来了一定的不确定性。因此，我希望通过开源方式分享这份笔记，一方面以中文书写为主，便于更多学习者阅读，同时保留关键英文术语；另一方面，我也希望整理出25–26学年的往年试题，以供后续同学复习参考。

**目前本仓库仅更新了往年试题部分，后续计划陆续补充试题解答及完整的笔记内容。**
